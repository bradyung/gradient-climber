# Gradient Climber

Gradient Climber is a C# game inspired by ideas from multivariable calculus. The goal of the game is to explore the terrain, follow gradient-based hints, avoid being misled by false summits, and find the true peak. This project was created for SDL 4 as a way to combine programming with math in a more interactive and creative format.

## Project Purpose

The purpose of this project was to build a game that connects programming with concepts from multivariable calculus. Instead of only solving math problems on paper, this project turns ideas like slopes, local maxima, and optimization into gameplay. The player moves through a terrain map and uses hints based on the gradient to search for the highest point.

## Features

- Terrain-based exploration
- Player movement across the map
- Fog-of-war effect that limits visibility
- Gradient hint arrows to guide the player
- False summits that can trick the player
- Peak reveal after winning
- Improved challenge and gameplay difficulty
- Visual effects that make the world feel more alive

## What I Learned

Through this project, I learned more about building a larger C# program with multiple parts working together. I also learned how multivariable calculus ideas can be represented visually in a game. While developing Gradient Climber, I practiced problem solving, debugging, improving game difficulty, and refining mechanics based on testing.

Some specific improvements I worked on included:
- Making the fog-of-war darker so the player could not easily see outside the visible area
- Improving the challenge so the game was not too easy
- Adjusting the hint system so it would not incorrectly lead players to false summits
- Adding more polish to make the game feel more complete

## How to Run

1. Open the project in Visual Studio.
2. Build the solution.
3. Run the program.
4. Use the keyboard controls in the game window to play.

## Controls

- Move the player using the keyboard
- Follow the hint arrows carefully
- Avoid false summits
- Reach the true peak to win
